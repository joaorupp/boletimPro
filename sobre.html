1. Identidade do Projeto

Nome Boletim Pro (MyGradeCalc)
Slogan "Simule suas médias e organize suas notas escolares em segundos."

2. O Problema e a Solução

  • Problema

    Alunos frequentemente têm dificuldade em calcular rapidamente suas médias finais ou prever quanto precisam tirar na próxima prova, ficando perdidos em meio a várias matérias.

  • Solução

    Uma interface web simples onde o aluno cadastra a matéria e suas notas. O sistema calcula a média automaticamente e mostra visualmente se ele está "Aprovado" ou "Reprovado".

3. Funcionalidades (JavaScript)

  • Entrada de Dados: Formulário para Disciplina, Nota 1 e Nota 2.
  • Lógica Principal: Cálculo de média aritmética (N1 + N2) / 2.
  • Validação: Identificação automática de status (Aprovado/Reprovado).
  • Interatividade: Adição dinâmica na tabela e botão de excluir item.
  • Persistência: Uso de Vetores (Arrays) na memória RAM (sem banco de dados).

4. Estrutura de Dados (Modelo)

Exemplo do objeto JSON armazenado no vetor:

{
  id: 167890123,
  disciplina: "Matemática",
  nota1: 8.5,
  nota2: 6.0,
  mediaFinal: 7.25,
  situacao: "Aprovado"
}
